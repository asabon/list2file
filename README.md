# list2file

## Input List file (list.xlsx)

| a | b | data | filename   | c |
|---|---|------|------------|---|
|   |   | 0x00 | file1.xlsx |   |
|   |   | 0x01 | file2.xlsx |   |
|   |   | 0x02 | file3.xlsx |   |

## Usage

> python list2file.py list.xlsx 4 3 2 3
